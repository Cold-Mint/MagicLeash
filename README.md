# MagicLeash
MagicLeash is a server plugin designed for PaperMc.
**Supported Version**: 1.21.x

## Features
This plugin allows players to right-click with a lead to leash other players.

1. Players can use shears to cut the lead that is restraining a player.
2. Leashed players cannot untie or break the lead knot themselves.
3. A single player can leash multiple players at once.
4. Leads are automatically removed when the leashed player disconnects from the server or dies.
5. Leads are automatically disconnected when the distance between players exceeds the limit.
